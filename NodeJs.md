# Node.js v0.12.7

## Ubuntu 15.04
```
$ curl --silent --location https://deb.nodesource.com/setup_0.12 | sudo bash -
$ sudo apt-get install --yes nodejs
```


## Windows 7

Update npm:
```
$ npm install npm -g
```

## Debian testing (Buster)

Install npm
```
sudo apt-get install npm
sudo npm install -g npm@next
```
